# Difference calculator Project

A difference calculator is a program that determines the difference between two data structures.

## Requirements:

```
Node.js v13+
```

## Installation:

````
$ git clone git@github.com:nikolaydlv/frontend-project-46.git
$ make install
$ sudo npm link
````
